# Установка тестового расширения

1. Откройте `chrome://extensions` или `edge://extensions`.
2. Включите режим разработчика.
3. Нажмите «Загрузить распакованное расширение» и выберите эту папку `browser-extension`.
4. Обновите страницу приложения.
5. Нажмите `iFrame`.

Расширение предназначено только для desktop Chrome/Edge. Мобильные Chrome и Safari такие расширения не поддерживают. Расширение получает логин и пароль только по запросу страницы и отправляет их непосредственно на `https://www.yclients.com/auth/login/1`.
